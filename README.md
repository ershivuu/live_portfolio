# 📌 About The Project

This project is my personal developer portfolio website designed to showcase my skills, projects, experience, and technical expertise in a modern and visually engaging way.

The portfolio is fully responsive and optimized for different screen sizes while maintaining smooth performance and clean UI/UX.

It serves as a central place to represent my professional work, development journey, and technical capabilities.

---

# ✨ Features

- ⚡ Modern & Responsive UI
- 🚀 Fast Performance with Vite
- 🎯 Smooth Navigation & Routing
- 📱 Mobile Friendly Design
- 🌙 Clean & Professional Layout
- 🧩 Reusable React Components
- 🔥 Optimized Build Structure

# 🚀 Getting Started

## Clone The Repository

```bash
git clone https://github.com/ershivuu/live_portfolio.git
```

## Navigate To Project Folder

```bash
cd live_portfolio
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

---

# 🏗️ Build For Production

```bash
npm run build
```
